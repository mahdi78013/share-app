# اپلیکیشن گلس‌شیر (GlassShare) 💎📦

یک ابزار استخراج و اشتراک‌گذاری APK فوق‌العاده زیبا، مدرن و با کارایی بسیار بالا برای سیستم‌عامل اندروید که با آخرین استانداردهای توسعه روز (Jetpack Compose، معماری MVVM و Material Design 3) به همراه جلوه‌های بصری شیشه‌ای بی‌نظیر (Glassmorphism) توسعه یافته است.

---

## 🎨 ویژگی‌های برجسته ظاهری و فنی

### ۱. رابط کاربری شیشه‌ای لوکس (Ultra-Premium Glassmorphic UI)
- بهره‌گیری از افکت شیشه‌ای نیمه‌شفاف هماهنگ با پس‌زمینه پویا مجهز به درخشش نئونی آبی (Electric Cyan).
- سایه‌های نرم و مرزهای نوری باریکِ دور قاب‌ها که حس عمق سه‌بعدی بی‌نظیری ایجاد می‌کنند.
- انیمیشن‌های بازخورد لمسی خیره‌کننده و جابه‌جایی‌های روان بدون افت فریم.

### ۲. موتور پیشرفته استخراج ایمن APK (High-Performance Chunked Extraction)
- **استخراج مستقیم سورس با کارایی بالا:** استخراج فایل‌های نصب برنامه‌ها بدون کوچک‌ترین اتلاف حافظه یا کراش.
- **بهینه‌سازی خودکار فضای ذخیره‌سازی:** سیستم هوشمند Pruning که تمام فایل‌های موقت و استخراج‌شده قدیمی‌تر از ۱۰ دقیقه را در زمان راه‌اندازی جدید یا رفرش لیست پاک می‌کند تا حافظه دستگاه کاربر بیهوده اشغال نشود. این منطق به صورت پس‌زمینه هماهنگ شده تا با فرآیند پردازش چندتایی (Batch Share) تداخل پیدا نکند.
- **قابلیت اشتراک‌گذاری خود برنامه (Self-Sharing):** اپلیکیشن قابلیت استخراج و به اشتراک‌گذاری مستقیم سورس خود را نیز داراست و کاربر می‌تواند با سرچ نام برنامه آن را برای ارسال انتخاب کند.

### ۳. مدیریت گروهی اپلیکیشن‌ها (Batch Extraction & Multi-Select)
- امکان انتخاب همزمان چندین نرم‌افزار به صورت دسته‌جمعی.
- فشرده‌سازی خودکار و خروجی یک‌پارچه با سرعت بسیار بالا جهت اشتراک‌گذاری همزمان چند فایل نصب تنها با یک لمس.

### ۴. سیستم تاریخچه منعطف به همراه شمارشگر زمانی پویا
- لاگ سه‌بعدی و امن با پایگاه داده محلی **Room** برای مشاهده تاریخچه اپلیکیشن‌های به اشتراک‌ گذاشته شده.
- **بروزرسانی زنده زمان (Dynamic Time Ticker):** استفاده از معماری نوین کوتین Flow پایدار متصل به چرخه حیات لایف‌سایکل گوشی برای نمایش عمر لحظه‌ای رویدادها (مثلاً "هم‌اکنون"، "۲ دقیقه قبل"، "۱۰ دقیقه قبل") بدون سربار اضافه به CPU و تداخل‌های چندگانه بر اثر اجرای موازی تسک‌ها.

## 🛠️ جزئیات معماری و فناوری‌های بکار رفته

این پروژه صد درصد با کاتلین توسعه یافته و از کتابخانه‌های مرجع و تاییدشده زیر استفاده می‌كند:

- **Jetpack Compose:** فریم‌ورک مدرن و اعلانی شیوه چینش اجزای صفحه اندروید هماهنگ با Material Design 3.
- **Room Database:** مدیریت لایه پایگاه داده محلی SQLite با قابلیت ناوبری غیرمخرب داده کاربری در صورت آپدیت نسخه دیتابیس (MIGRATION_1_2).
- **Kotlin Coroutines & Flow:** به کار بردن برترین الگوهای اجرای موازی نخ‌ها (Threading) و هندلینگ پیشرفته جریان داده‌های غیرهمزمان.
- **LruCache بهینه‌سازی شده:** کش امن آیکون‌ها بر پایه محاسبات واقعی حافظه Heap سیستم با محافظت سرریز عدد صحیح (`coerceIn`) در تلفن‌های بالارده و مدرن.
- **Keystore مجاز:** دارای امضای رسمی مجاز برای بیلدینگ و توزیع پایدار و امن نرم‌افزار.

## 🚀 راهنمای نصب و راه‌اندازی توسعه‌دهندگان

به منظور بیلد و توسعه اپلیکیشن در محیط محلی خود، به موارد زیر نیاز دارید:
- سیستم‌عامل با نسخه JDK 17
- اندروید استودیو نسخه Ladybug به بالا
- حداقل SDK اندروید: ۲۶ (Oreo) و هدف‌گذاری روی نسخه ۳۴ (Android 14)

### اجرای دستور بیلد از طریق ترمینال
برای همگام‌سازی گریدل و تولید پکیج نصب موقت دیباگ دستور زیر را در روت پروژه اجرا کنید:

```bash
gradle assembleDebug
```

```kotlin
val MIGRATION_1_2 = object : Migration(1, 2) {
    override fun migrate(db: SupportSQLiteDatabase) {
        // مهاجرت امن بدون پاک شدن لاگ‌ها و دیتاهای برنامه کاربر در صورت آپدیت فیلدها در آینده
    }
}
```

طراحی شده با عشق و تعهد به خلق والاترین تجربه‌های بصری و بهینه‌ترین سورس کدهای مهندسی نرم‌افزار. 💎🎨
